GUI Helper allows you to configure after installation.
![GUI Helper](images/guihelper.png)
!!! info
	When you click the GUI Helper shortcut, it will download and run it from GitHub each time. If you don't need GUI Helper, simply delete the shortcut to remove it.

!!! info
	If you removed the shortcut, you can access the GUI Helper with the following command:
	```bash
	bash <(curl -sSL https://raw.githubusercontent.com/unlbslk/arch-deckify/refs/heads/main/gui_helper.sh)
	```
